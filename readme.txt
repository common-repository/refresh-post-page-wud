== Description ==

= FREE Premium Plugin with updates, PRO support and new features! =
* If you use WordPress by example to display some variable content which is available on a page or post, this plugin will refresh this content every X seconds, where X is the quantity.
* A sample could be a page/ post which refresh the content each X seconds of an information board, or to display current news, currency info, ... ...
* Guaranteed without any limitation.
* Can be used on any page or post !
* Discover and enjoy our free plugins [here](https://wud-plugins.com/).
*
= About Refresh Post Page WUD =
* Easy to use auto refresh field per post and/or page, no extra code required
* Per post and/or page you can enter an amount of seconds, when the content has to refresh.
* Sample: the value 300 is equal on 5 minutes.
* Read our online manual [here](https://wud-plugins.com/refresh-post-page-wud-how-to-use/).

== Installation ==

= Automatic installation: =

* The simplest way to install is to click on 'Plugins' then 'Add' and type 'Refresh Post Page WUD' in the search field.

= Manual installation 1: =

1. Login to your website and go to the Plugins section of your admin panel.
1. Click the Add New button.
1. Under Install Plugins, click the Upload link.
1. Select the plugin zip file (refresh-post-page-wud.x.x.x.zip) from your computer then click the Install Now button.
1. You should see a message stating that the plugin was installed successfully.
1. Click the Activate Plugin link.

= Manual installation 2: =

1. You should have access to the server where WordPress is installed. If you don't, see your system administrator.
1. Upload the plugin zip file (refresh-post-page-wud.x.x.x.zip) up to your computer and unzip it somewhere on the file system.
1. Copy the "refresh-post-page-wud" folder into the /wp-content/plugins directory of your WordPress installation.
1. Login to your website and go to the Plugins section of your admin panel.
1. Look for "Refresh Post Page WUD" and click Activate.

= Usage =
* Enter the number of seconds in the provided Refresh Post/Page WUD field.
* Sample: the value 300 is equal on 5 minutes.

== Frequently Asked Questions ==

= How can i activate the Auto Refresh? =
* Enter the number of seconds in the provided Refresh Post/Page WUD field.
* If there is no page refresh, you have entered a wrong value. The value must be numeric.
